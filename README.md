# portfolio-website--2<br>
📌 Portfolio Description

This is a personal portfolio website created using HTML, CSS, and JavaScript.
The website has a header that shows my name Riya Gupta and a short tagline.

A navigation bar is added to move between Home, About, Projects, and Contact sections.

The Home section explains my hobby, coding.
It includes a list of coding benefits where new items can be added and deleted using JavaScript.

The About section gives a short introduction about my interest in web development.

The Projects section shows three sample projects displayed using Flexbox.

The Contact section contains a form with name, email, and message fields.
When the form is submitted, a success message is shown.

The footer includes links to LinkedIn and GitHub.
It also displays the current date and time using JavaScript.

This portfolio is simple, responsive, and suitable for beginners learning web development.
